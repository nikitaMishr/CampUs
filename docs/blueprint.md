# **App Name**: CampUs

## Core Features:

- Secure Role-Based Authentication: Users and Admins can sign in via a secure login page featuring clear role selection, with authentication and user roles persisting across sessions.
- Dynamic Canteen Services: Users can view food items, add to cart, place orders, manage quantities, and track kitchen status. Admins can manage the full menu, including adding, editing, and deleting items, updating prices, availability, and kitchen statuses.
- Streamlined Issue Reporting & Tracking: Users can submit and track campus issues with optional photo uploads. Admins can efficiently view all reported issues and update their statuses.
- AI Issue Triage Tool: An AI assistant helps Admins by providing automated summaries and suggesting relevant categories for newly reported issues, expediting the triage process.
- Comprehensive Lost & Found Hub: Users can easily post details for lost or found items with image uploads. Admins can view all submitted items, facilitate manual matches, and delete inappropriate posts.
- Personalized Event Discovery & Notification System: Users can filter events by department, view detailed event information, and receive real-time notifications for relevant events. Admins can create, edit, delete events, and send targeted notifications to selected departments.
- Interactive Campus Navigation Map: Users can explore a dynamic campus map, view their current location, select a destination, and receive estimated walking times with visualized routes. Admins can manage building markers, including names, coordinates, and visibility.

## Style Guidelines:

- Primary Color: Deep Blue (#1E3C72). Chosen for its sophisticated and stable presence, evoking the reliability of a 'Smart Campus Hub' in a modern context.
- Background Color: A subtle, very dark blue-grey (#15191E). This deep, desaturated hue provides an ideal dark-mode foundation, allowing UI elements with glassmorphism to pop with elegance.
- Accent Color: Neon Cyan (#00F5FF). This vibrant and high-contrast color is perfect for highlighting key interactive elements, notifications, and animated effects, aligning with the 'neon' aesthetic and creating an energetic, digital feel.
- Body and Headline Font: 'Poppins' (sans-serif). Selected for its geometric structure and precise, contemporary appearance, which complements the sleek and avant-garde style of the application.
- Modern and clean line-art or glyph-based icons are recommended. These icons will feature subtle glowing effects when interactive, perfectly complementing the dark theme and glassmorphism elements.
- The layout is designed for modern responsiveness, featuring glassmorphism login cards centered on blurred backgrounds, dynamically arranged feature cards, and an accessible floating circular map button for seamless navigation.
- Smooth and elegant animations are paramount, including seamless transitions, graceful slide-down/expand effects, 'neon glow' hover states, subtle pulsing for interactive elements, and fluid fade animations for modal closures.
